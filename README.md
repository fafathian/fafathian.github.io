Selamat datang di repositori website portofolio saya! Website ini merupakan ruang digital untuk menampilkan rekam jejak profesional, keahlian, dan berbagai proyek yang telah saya kerjakan sebagai Software QA Tester dan pengembang aplikasi.

🛠 Tech Stack
Website ini dibangun dengan pendekatan static-first yang ringan, cepat, dan responsif:

Markup: HTML5

Styling: Tailwind CSS

Deployment: GitHub Pages

🚀 Tentang Proyek
Proyek ini dibuat untuk menyajikan informasi portofolio secara bersih dan terstruktur. Fokus utama pengembangan adalah pada antarmuka yang modern, aksesibilitas yang baik, serta pengalaman pengguna yang mulus di berbagai perangkat.
